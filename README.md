## What

Download all [soundcloud](https://soundcloud.com/) tracks

```
---------------------------------
Original userscript by Technowise
---------------------------------
Pub Mod: THRSH - 9:17AM 1/14/2014
---------------------------------

  - Resolved URL on click for faster page render

```
